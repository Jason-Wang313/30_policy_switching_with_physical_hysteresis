# Novelty Decision

Chosen thesis: **Policy switching in physical contact should be treated as a hysteretic control problem with measurable physical switch cost.**

Why this is the strongest idea:
- It changes the central mechanism from instantaneous policy choice to path-dependent switch accounting.
- It is directly relevant to adaptive robot control and contact-rich manipulation.
- It is distinct from generic dwell-time and chatter suppression because the switch itself is modeled as a physical event with cost.

Rejected weaker directions:
- bigger model
- better data
- new benchmark only
- add uncertainty
- add active learning
- add verifier
- combine two existing modules
- use an LLM as planner
- use reinforcement learning
